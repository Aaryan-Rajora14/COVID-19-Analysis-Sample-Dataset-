# COVID-19 Dashboard - Power BI

An interactive and visually appealing Power BI dashboard for monitoring and analyzing COVID-19 data worldwide.

![IMG 1](https://github.com/user-attachments/assets/824c6569-a22b-41e2-b740-edf7708843ee)

## 📊 Features

- **Global Overview**: Total cases, deaths, recoveries, and active cases
- **Time Series Analysis**: Daily/weekly trends and growth rates
- **Country-wise Comparison**: Top affected countries, vaccination progress
- **Regional Insights**: Continent and country-level breakdowns
- **Key Metrics**: Case fatality rate, recovery rate, vaccination coverage
- **Interactive Filters**: Date range, country/region slicers, and drill-through
- **Professional Design**: Modern theme with consistent color scheme and responsive visuals

## 🛠️ Technologies Used

- **Power BI Desktop**
- **DAX** (Data Analysis Expressions)
- **Power Query** for data transformation
- Custom theme (CY24SU10)



## 🚀 How to Use

1. Download the `Covid 19 Dashboard.pbix` file
2. Open it using **Power BI Desktop** (Free)
3. Refresh the data if needed (if connected to live sources)
4. Interact with slicers, filters, and drill-down options

## 📈 Dashboard Pages

- **Overview** – Global summary and key KPIs
- **Trends** – Time-based analysis and forecasts
- **By Country** – Detailed country comparison
- **Vaccination** – Progress and coverage analysis
- **Regional Analysis** – Continent-wise insights

## 📝 Data Source

COVID-19 data from reliable public sources (Kaggle).  
*Note: Last data refresh date is visible inside the report.*
